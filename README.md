## How to run locally
1. Clone/copy the `portfolio/` folder.
2. Open `index.html` in your browser (no build step required).
3. Replace placeholder images in `assets/img/` and update links.

## To deploy (GitHub Pages)
- Create a repo (e.g., `portfolio`), push the folder, then enable GitHub Pages from `main` branch -> `/ (root)`.
